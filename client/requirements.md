## Packages
framer-motion | Animations and page transitions for premium feel
recharts | Admin dashboard analytics
i18next | Internationalization core
react-i18next | React bindings for i18n
i18next-browser-languagedetector | Automatic language detection

## Notes
- The application uses Replit Auth for user authentication (`/api/login` and `/api/logout`).
- Dynamic images are supported; placeholders use Unsplash with descriptive comments.
- Uses Urbanist Google Font for a modern, high-conversion aesthetic.
- Includes i18n setup with RTL support for Arabic (AR).
